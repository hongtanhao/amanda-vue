# 如果想看就撇一眼又无妨
## 为什么会`[AmandVue]`这个项目？
> amanda，英文意是`值得被爱的`，至于vue，一直是我最喜欢的前端开源框架之一，近几年大大小小项目都有在使用，也正因此才有了这个工程，由我一人开发，并且本人计划贡献全部源码，主要以学习研究为目的，不打算商用
## 组件特点
+ 简单，除了暴露出必要的接口，其他都已约定好
+ 高雅，约定大于配置，既灵活又高效
+ 实用，大多都来自实际项目所累，也是高频使用组件
## 主要技术栈
+ vue
+ nodejs
+ webpack
