# drag-container 可拖动可放大缩小的容器

## 在 vue 的单页面中这样用：

```bash
npm install amav --S
```

在项目入口文件中导入
`main.js`

```javascript
import Vue from 'vue'
import {AmaDragContainer} from 'amav'
Vue.use(AmaDragContainer)
...
```

## 基础用法

```html
<ama-drag-container></ama-drag-container>
```
